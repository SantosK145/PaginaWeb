# Pagina de Ghost of Tsushima
## Proyecto Web

Este repositorio contiene un proyecto web desarrollado con HTML, CSS y JavaScript

## Requisitos previos

Antes de ejecutar el proyecto, asegúrate de contar con:
* Un navegador web moderno (Google Chrome, Firefox, Edge, etc.)
* Git instalado (opcional, pero recomendado)
* Un editor de código como Visual Studio Code (opcional)

## Instalación
Puedes obtener el proyecto de dos maneras:
### Clonar el repositorio
git clone https://github.com/USUARIO/NOMBRE_DEL_REPOSITORIO.git
Luego accede a la carpeta del proyecto:
cd NOMBRE_DEL_REPOSITORIO

## Ejecución local
Proyecto estático (HTML/CSS/JS)
1. Abre la carpeta del proyecto
2. Da doble clic al archivo index.html
3. El proyecto se abrirá en tu navegador

## GitHub Pages
El proyecto está publicado usando GitHub Pages y puede verse en la siguiente URL:
https://USUARIO.github.io/NOMBRE_DEL_REPOSITORIO/

## Cómo se desplegó
1. Ir a Settings del repositorio
2. Seleccionar Pages
3. Elegir la rama main y la carpeta /root
4. Guardar los
